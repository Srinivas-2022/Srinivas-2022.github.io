SECTION 01

Setup & Architecture

How Claude connects to Oracle 23ai Free Edition through SQLcl's MCP server — zero-middleware, direct tool invocation.

01
Install Oracle 23ai Free Edition
Run Oracle 23ai Free on Docker or local install. Default listener on port 1521, SID FREE. Connect as SYSTEM or create a dedicated DBA account.
02
Download & Configure SQLcl 24.x
Download SQLcl from Oracle. Ensure Java 11+ is on PATH. SQLcl ships with a built-in MCP server via mcp start command — no extra install needed.
03
Launch SQLcl MCP Server
Connect SQLcl to Oracle 23ai, then start the MCP server endpoint. Claude Desktop or API clients discover tools via the standard MCP manifest at the configured port.
04
Configure Claude MCP Client
Add the SQLcl MCP server URL to Claude's tool configuration. Claude now has access to sql_query, describe_table, list_objects, and other SQLcl tools natively.

🔍
Dry-Run Preview
Claude always SELECTs affected rows before destructive DML — showing before/after values and row counts so you confirm scope before changes land.
💾
Savepoint Management
Claude wraps DML in SAVEPOINT/ROLLBACK logic automatically when requested — giving granular undo control without full transaction rollback.
✅
Constraint Validation
Before any INSERT/UPDATE, Claude validates FK parents, sequence values, UNIQUE constraints, and NOT NULL requirements — preventing ORA- errors proactively.
⚠️
23ai Free Limits
Oracle 23ai Free caps at 2 CPUs, 2GB RAM, 12GB storage. DML on large datasets may be throttled. AUTONOMOUS_TRANSACTION and DBMS_LOCK have restricted access for non-SYSDBA users.

SECTION 05

Oracle 23ai New Features

23ai ships game-changing capabilities that Claude can leverage directly — JSON Relational Duality, Vector Search, True Cache, and more.

🔄
JSON Relational Duality Views
NEW in 23ai
One object, two faces. Expose relational tables as JSON documents — read and write via either SQL or REST/JSON without duplication or sync overhead. Claude can query both representations through a single MCP tool call.
🔍
AI Vector Search
NEW in 23ai
Native VECTOR datatype and VECTOR_DISTANCE() function. Store embedding vectors alongside relational data — enabling hybrid semantic + keyword search with a single SQL query. Claude can generate embeddings and query them directly.
📊
SQL Domains
ENHANCED
Define reusable column metadata — format, display, validation rules — once in a DOMAIN and apply across any column. Claude can inspect domain definitions via ALL_DOMAINS to understand data semantics without hardcoded documentation.
⚡
True Cache
NEW in 23ai
A read-only in-memory cache tier — eliminates the need for separate Redis/Memcached layers. Claude can check True Cache hit rates via V$TRUE_CACHE and recommend which objects to cache based on V$SQL hot-spot analysis.
🎯
Property Graphs
ENHANCED
SQL/PGQ — query graph relationships natively in SQL using GRAPH_TABLE() syntax. No separate graph database needed. Claude can traverse graph paths on relational data — ideal for org hierarchies, supply chains, and dependency mapping.
🔒
Schema Privileges
ENHANCED
Grant a single privilege on an entire schema — GRANT SELECT ANY TABLE ON SCHEMA hr TO app_user. Eliminates per-table GRANT sprawl. Claude can audit schema-level privilege assignments through ALL_SCHEMA_PRIVS.
